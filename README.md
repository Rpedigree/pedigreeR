pedigreeR
=========
The ```pedigreeR``` package offers a suit of functions for pedigree analyses, including: sorting and editing pedigree data, computing inbreeding, additive relationships and functions of it (e.g., cholesky decomposistions, inverse of a numerator relationship matrix, etc.). The package was originally co-developed by Douglas M. Bates and Ana I. Vazquez and was incorporated in the [lme4]() R-package.
